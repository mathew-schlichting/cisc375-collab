The Project:

ChatBlocs is a collaborative video, text message and drawing app. A user has to join a room to participate, selecting a color and username as they do so (the color will be used to identify this user in the text message box). Each room is limited to four users. The users in each room are able to start a video call among themselves. In addition, they can each see what the others write in the chat box as well as what everyone draws on the canvas.

ChatBlocs uses WebRTC to make video and audio connections between users. It also uses SocketIO to communicate between the users in each room, facilitate collaborative drawing, and signal between the pages. It also uses the jQuery and AngularJS frameworks to navigate between rooms and facilitate user interaction with the page.
